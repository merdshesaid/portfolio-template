# MERDWeb Portfolio

A modern, lightweight portfolio website template built with **HTML5**, **CSS3**, and **JavaScript**. Designed for developers, freelancers, and creatives who want a fast, responsive, and easily customizable personal portfolio without relying on frameworks or build tools.

---

## Overview

**MERDWeb Portfolio** is a static website template focused on simplicity, performance, and maintainability. It provides a clean foundation for showcasing projects, skills, experience, and contact information while keeping the codebase organized and easy to extend.

## Features

- Modern and responsive design
- Mobile-first layout
- Semantic HTML5 structure
- Vanilla JavaScript (no frameworks)
- Clean, maintainable CSS
- Cross-browser compatibility
- Lightweight and fast
- Easy to customize
- SEO-friendly structure
- Ready for deployment on any static hosting platform

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Styling & responsive layout |

## Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/merdweb-portfolio.git
```

Navigate into the project directory:

```bash
cd merdweb-portfolio
```

Open the project in your preferred editor:

```bash
code .
```

Run the project using the **Live Server** extension for VS Code or simply open `index.html` in your browser.
